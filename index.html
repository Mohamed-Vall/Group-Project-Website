Friendly Neighbor Forecast

Welcome to our neighborly weather forecast

A small web project made by your classmates Christos Maragos, Costantinos.
Kores and Mohamed Mohamed Vall. This project is a small one, made with
testing out unix and other web-based language's functionalities.
It may not look like much, but we've learned a great deal while working on this project and hope you'll learn to appreciate the work
we put into it.

Functionalities

Although most of the work for this project is happening "behind the scenes"
(ex: launching this project on a web server, keeping said server open and
constantly updating, etc.), we've decided to build a few functionalities
with the hopes of making a robust project every can enjoy. Firstly,
we've made not one but two (!) variants of a weather forecasting program
Those being the Single and Multiple daily forecast.
With that said, we would like to present to you the fruits of our labor.

Enjoy!
